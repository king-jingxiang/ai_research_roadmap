# AI Research Roadmap

[English](README.md) | [中文](README_ZH.md)

> **洞察论文脉络，推演技术未来。**
> 本项目是一款自动化的技术演进追踪工具，通过集成 arXiv 与 Semantic Scholar 数据源，利用大模型（LLM）深度解析论文内容，并生成可视化的技术演进路线图，协助研究者从浩如烟海的文献中快速理清领域脉络并预判未来方向。

---

## 🌟 核心功能

* **多源论文检索**：集成 **arXiv** 与 **Semantic Scholar** API，精准覆盖主流学术资源。
* **关键内容提取**：利用大模型对论文核心贡献、技术点进行结构化提取，告别繁琐的原文初筛。
* **技术演进路线生成**：基于长文本总结技术，将特定领域的关键论文按时间线串联，自动生成技术演变逻辑。
* **未来趋势推演**：基于已有技术路径，利用 LLM 进行深度推测，预测该领域可能的后续研究热点与演进方向。
* **交互式 Web 界面**：直观的展示页面，支持按领域、评测任务（如 SWE-bench）进行定向分析与存档管理。

---

## 🛠️ 技术架构

* **数据层**：arXiv API, Semantic Scholar API
* **模型层**：LLM (如 GPT-4, Claude 等) 负责提取、总结与推演
* **处理层**：长文本处理工作流（Long-context Summarization）
* **前端展示**：基于 Web 的交互式仪表盘

### 典型用例：SWE-bench 演进分析

## 📅 开发计划 (Roadmap)

* [ ] 支持更多论文数据库（如 ACL, IEEE）。

* [ ] 增加多智能体（Multi-agent）协同推演模式。
